# 🏨 ITC Hotels Business Intelligence Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze the business performance of ITC Hotels. It provides insights into revenue, occupancy, ADR, RevPAR, room category performance, booking trends, and cancellation analysis to support data-driven decision-making.

---

## 🎯 Business Objectives
- Analyze revenue performance
- Monitor occupancy trends
- Evaluate ADR and RevPAR
- Identify top-performing hotels and room categories
- Analyze booking behavior
- Understand cancellation patterns
- Provide actionable business recommendations

---

## 📊 Key KPIs
- Total Revenue
- Occupancy Rate
- Average Daily Rate (ADR)
- Revenue Per Available Room (RevPAR)
- Cancellation Rate
- Refund Amount
- Average Length of Stay (ALOS)
- Booking Lead Time

---

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- Microsoft Excel

---

## 📈 Dashboard Features
- Revenue Analysis
- Occupancy Analysis
- Room Category Performance
- Booking Insights
- Cancellation & Lost Revenue Analysis

---

## 💡 Key Insights
- RT2 generated the highest revenue.
- Weekend occupancy was consistently higher than weekdays.
- Revenue experienced a decline in June before recovering.
- Cancellation rate remained close to 25%.
- Revenue was concentrated among top-performing hotels.

---

## 🚀 Recommendations
- Improve performance of low-performing properties.
- Implement dynamic pricing strategies.
- Increase weekday occupancy through targeted offers.
- Encourage early bookings.
- Reduce cancellations using predictive analytics.
